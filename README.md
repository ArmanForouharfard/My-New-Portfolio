## ✨ Features

- **Cinematic dark theme** — `#0a0a0c` base with `#f07a2a` orange accent system
- **CSS custom properties** — full design token architecture, one-line theme changes
- **Scroll-triggered animations** — IntersectionObserver-powered fade-up entrance effects
- **Typewriter effect** — rotating role titles in the hero section
- **Animated proficiency bars** — skill levels animate in on scroll
- **Project filter system** — JavaScript-powered category filtering on the projects page
- **Fully responsive** — fluid typography with `clamp()`, mobile hamburger nav, adaptive layouts
- **Semantic HTML** — proper use of `<nav>`, `<section>`, `<aside>`, `<footer>`, `<article>`
- **No dependencies** — zero npm, zero frameworks, zero build step
- **GitHub Pages ready** — deploy instantly with no configuration
